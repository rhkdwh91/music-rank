# music-rank

music-rank

# 패키지 다운로드

npm install

# 실행

npm run start

# 추가 사용 lib

axios, react-router-dom, lodash, typescript v4

> 추가적인 별도의 상태관리 및 스타일 라이브러리는 사용하지 않았습니다.

# 폴더구조

아토믹 디자인 패턴으로 기능적인 부분을 hook으로 분리하여
최대한 컴포넌트 및 기능을 재활용 할수 있게 만들었고
page 컴포넌트에서 봤을때 해당 페이지의 전체적인 흐름을 한눈에 볼수 있도록 설계하였습니다.

# 개발환경

Node.js v20.11.0
